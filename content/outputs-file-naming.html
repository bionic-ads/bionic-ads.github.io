# File Naming ## Media Plan Exports - *Org* - *Advertiser* - *Campaign* - Google Analytics URLS.xls - *Org* - *Advertiser* - *Campaign* - Media Authorization.xls - *Org* - *Advertiser* - *Campaign* - Media Plan.xls - *Org* - *Advertiser* - *Campaign* - Sizmek Placement Import.xls - *Org* - *Advertiser* - *Campaign* - Program IO Schedule.xls ## Flowchart - *Advertiser* - *Campaign* - Flowchart.xls - *Advertiser* - *Campaign* - Flowchart.pdf ## Allocations - *Advertiser* - *Campaign* - Allocations - *Report Name*.pdf **EXAMPLE:** Tesla - Model X SUV Launch - Allocations - Channels.xls ## Performance - **PDF:** *Advertiser* - *Campaign* - Performance - *Metric Name* - *Metric View*.pdf - **Excel:** *Advertiser* - *Campaign* - Performance - *Metric Name* - *Metric View*.xls **EXAMPLE:** ## Usage Heatmap - Org Name - Bionic Usage - mm-dd-yyyy - mm-dd-yyyy.xls **EXAMPLE:** Merkle - Bionic Usage - 11-27-2016 - 02-18-2017.xls
