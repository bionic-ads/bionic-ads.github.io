# Buttons ## Usage Buttons can be rendered as inputs (example: form submission buttons) or as links styled as buttons (example: Add Placement). In both cases they should be styled to look the same. ## Primary Action Add Placement 
    <a class="blue-button">Action</a>
## Primary Disabled 
    <input class="blue-button" disabled="disabled" value="Apply" />
## Secondary Action Save As... 
    <a class="gray-button">Save As...</a>
## Grid Action Buttons Grid action buttons are controls used above grids of data. They are typically include an icon with a text label. Grid action buttons also have a grayed-out state (some actions are only valid when an item is selected). ![Grid Buttons](images/content/elements/grid-buttons.png)
