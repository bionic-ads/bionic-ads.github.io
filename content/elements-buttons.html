Buttons

Buttons can be rendered as inputs (example: form submission buttons) or as links styled as buttons (example: Add Placement). In both cases they should be styled to look the same.

Primary Action

Add Placement 
    <a class="blue-button">Action</a>

Primary Disabled

    <input class="blue-button" disabled="disabled" value="Apply" />

Secondary Action

Save As... 
    <a class="gray-button">Save As...</a>
