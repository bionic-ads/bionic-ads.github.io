# Radio Buttons ## Usage Select one item from a list of items. Use for shorter lists where viewing all choices would be beneficial
## Specifications - Labels should be clickable and have cursor: pointer style on hover - A default option must always be selected ## Implementation 
    <div class="bionic-radio">
    <input id="item1" name="items" type="radio" value="item1" checked />
    <label for="item1">
        <span></span>
        Item 1
    </label>
</div>
