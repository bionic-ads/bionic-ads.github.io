Icons

We use the Font Awesome library as our primary resource for icons. Any custom icons should ideally be created as SVG files using Font Awesome icons as a starting point.

Custom Icons

In certain cases, we need specialized icons to represent actions or parts of the system. In these cases, we create SVG icons. Ideally any custom icons are built from Font Awesome parts.
